## 📌 Description

This project is a graphical ATM simulation built using Python and Tkinter.
It allows users to simulate basic banking operations such as PIN authentication, balance checking, and cash withdrawal.

---

## 🚀 Features

* Secure PIN verification system
* Account type selection (Savings / Current)
* Balance inquiry
* Cash withdrawal with conditions:

  * Amount must be in multiples of 500
  * Amount must not exceed available balance
* Error handling for invalid inputs
* Interactive GUI with buttons, labels, and entry fields
* Timed transitions using Tkinter `.after()` method

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI library)

## 📊 Functionality Flow

1. User inserts card
2. Enters PIN
3. Selects account type
4. Chooses operation:

   * Check Balance
   * Withdraw Money
5. System processes request and displays result
